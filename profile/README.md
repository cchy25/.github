## 🌱 About Qstart

**Qstart**는 창업을 결심한 청년이 스스로 자신의 위치를 진단하고, 맞춤형 로드맵과 정책을 통해 정책 의존이 아닌 전략적 활용이 가능한 창업 여정을 설계하도록 돕습니다.


- 📊 **데이터 크롤링 및 정제 파이프라인**
- 🤖 **LLM 기반 데이터 정제**
- ⏱ **Airflow를 통한 자동화된 워크플로우**
- 🧠 **사용자 맞춤 지원사업 추천 시스템**
